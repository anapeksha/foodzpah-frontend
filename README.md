<!-- GETTING STARTED -->
## Getting Started

Instructions on how to set up the project.

### Prerequisites

[Node.js](https://nodejs.org) v16.13.2.

  ```sh
  corepack enable
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/anapeksha/foodzpah-frontend.git
   ```
3. Install NPM packages
   ```sh
   yarn install
   ```
4. Configure environment vairables  in `.env`
